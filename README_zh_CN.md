# 功能
一个SiYuan插件，可以删除菜单/底栏/顶栏中你不使用的按钮。  
现阶段支持移除顶栏指定按钮，侧栏指定按钮，右键菜单项内容。     
本插件在MIT协议下开源，没有联网行为，也没有从远端fetch恶意代码的能力。
